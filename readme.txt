== Installation ==

**Method 1: WordPress Admin (Recommended)**
1.  Go to **Plugins > Add New** in your WordPress dashboard.
2.  Search for **Ultimate Spin Wheel**.
3.  Click **Install Now** and then **Activate**.
4.  Navigate to the **Ultimate Spin Wheel** menu to create your first campaign!

**Method 2: Manual Upload**
1.  Download the plugin ZIP file.
2.  Go to **Plugins > Add New > Upload Plugin**.
3.  Upload the ZIP file and activate it.

== Frequently Asked Questions ==

= Does the spin wheel work on mobile devices and tablets? =
Yes! The Ultimate Spin Wheel is fully responsive and touch-optimized for all mobile devices. Mobile users can spin using natural swipe gestures or simple tap controls. It automatically adjusts to any screen size (from 320px to 4K) to ensure a seamless experience. Mobile conversion rates typically range from 8-12%, which is 3-5x better than traditional signup forms.

= How do I prevent users from spinning the wheel multiple times? =
The plugin includes comprehensive fraud prevention to stop repeat spins. It uses **Cookie tracking** (works for 95% of users), **IP address limiting**, and **Email verification** to ensure fairness. The Pro version adds **Device fingerprinting** for advanced tracking even if cookies are cleared. You can configure limits to allow spins once per session, day, week, or campaign.

= Can I control the probability of winning each prize? =
Absolutely! Each prize slice has independent probability controls from 0-100%. You can set the "Grand Prize" to 0.5% (rare) and a "Small Discount" to 40% (common). You can even set slices to 0% for "Bad Luck" results. The Pro version allows dynamic probability adjustments based on inventory or cart value to protect your margins while maximizing excitement.

= Which email marketing platforms can I integrate with the spin wheel? =
We support direct integration with major platforms like **Mailchimp**, **MailerPoet**, **Constant Contact**, and **GetResponse**. Specifically, the Pro version offers universal connections via **Zapier webhooks**, allowing you to connect with over 3,000 apps (Google Sheets, Salesforce, Slack, etc.). Emails sync automatically with custom tags based on the prize won.

= Will adding a spin wheel slow down my WordPress site? =
No. The plugin is engineered for performance with a tiny footprint (under 50KB compressed). It uses **Lazy Loading** so assets only load when the trigger condition is met. There is no jQuery dependency (vanilla JS only), and logic is pre-calculated to ensure 90+ PageSpeed scores. It has zero impact on pages where the wheel is not active.

= Can I use the spin wheel without WooCommerce? =
Yes! While the WooCommerce integration is powerful, you can use the wheel for **Lead Generation** (ebooks, webinars), **Service Businesses** (appointments, consultations), or **SaaS** (free trials). Use the "Custom Link" prize type to redirect winners or "Text Message" to display secret codes. It is perfect for any business focused on growing an email list.

= How does the automatic WooCommerce coupon creation work? =
The plugin enables a "Hands-Free" coupon workflow. When a user wins, a **unique single-use coupon** is generated instantly with a unique code (e.g., SPIN-2026-X9M). This coupon is automatically applied to their cart via a "Shop Now" link. You can set expiration dates, minimum spend limits, and product exclusions directly within the wheel settings.

= What can the AI Assistant do to improve my spin wheel performance? =
The **AI Assistant** (Pro) analyzes your campaign data 24/7 to provide actionable insights. It can tell you which days convert best ("Tuesday is up 23%"), suggest optimization for specific triggers ("Exit-intent converts 2.1x better on product pages"), and identify which prizes lead to actual purchases. It allows you to query your data using natural language.

= Is the spin wheel plugin GDPR and privacy law compliant? =
Yes. We include built-in tools for **GDPR and CCPA compliance**. This includes a customizable privacy checkbox before spinning, compatibility with cookie consent plugins, and tools for data export/erasure ("Right to be Forgotten"). All data is stored securely in your WordPress database, giving you full control.

= What happens if someone wins a coupon but doesn't use it? =
The Pro version features an **Abandoned Coupon Recovery** system. You can set up an automated email sequence to remind winners: an immediate "Here is your code" email, followed by a "24-hour reminder", and finally a "Last Chance" urgency email. This strategy typically recovers 30-40% of lost sales.

= Can I schedule seasonal campaigns like Black Friday or Christmas sales? =
Absolutely. The Pro version includes a **Campaign Scheduler**. You can set precise start and end dates (e.g., auto-activate Nov 24, deactivate Nov 27) so you don't have to work holidays. You can also create recurring campaigns (e.g., "Weekend Flash Sale") and use seasonal themes like Halloween or Christmas to boost engagement.

= Can I run spin wheel campaigns in multiple languages? =
Yes! The plugin is fully translation-ready and supports **WPML**, **Polylang**, **Loco Translate**, and **TranslatePress**. You can create separate wheels for different languages or translate string elements (buttons, messages, prize labels) to allow a truly global marketing strategy from a single installation.

= How do I A/B test different wheel designs and offers? =
Optimization is key. You can perform A/B testing by duplicating a campaign and changing one variable (e.g., "Neon Theme" vs. "Classic Gold", or "10% Off" vs. "Free Shipping"). Run them sequentially or assign them to different pages to see which variation yields a higher email capture rate and redemption value.

= How long does it take to set up a spin wheel campaign? =
You can launch a professional campaign in **under 5 minutes**. Simply install, choose one of the 10+ pre-designed themes, configure your prizes, and click Publish. The intuitive visual builder requires no coding. We provide templates for "Lead Magnet", "Cart Recovery", and "Flash Sales" to get you started instantly.

= Do you offer support if I get stuck? =
Yes! We pride ourselves on top-tier support. Documentation is available directly within the plugin via the Help Center. Pro users receive priority email support from our developer team to assist with custom setups, CSS styling, or complex integrations.
