== Installation ==

**Quick Setup in Under 5 Minutes!**

**Method 1: WordPress Admin (Recommended)**
1. Navigate to `Plugins > Add New` in your WordPress dashboard
2. Search for "Ultimate Spin Wheel"
3. Click "Install Now" and then "Activate"
4. Go to `Ultimate Spin Wheel` in your dashboard sidebar
5. Follow the setup wizard to create your first wheel

**Method 2: Manual Upload**
1. Download the plugin ZIP file
2. Upload to `/wp-content/plugins/ultimate-spin-wheel` directory
3. Activate through the 'Plugins' menu in WordPress

**Method 3: FTP Upload**
1. Unzip the plugin file
2. Upload the `ultimate-spin-wheel` folder to `/wp-content/plugins/`
3. Activate through WordPress admin

== Frequently Asked Questions ==

= Can I control winning probabilities? =
Yes! You have granular control over individual prize chances, overall win/lose ratios, time-based changes, and user-specific rules.

= How can Ultimate Spin Wheel increase my sales? =
Ultimate Spin Wheel leverages gamification psychology to boost conversions by 200-300%. It captures emails, reduces bounce rates, prevents cart abandonment, and creates engaging experiences that turn visitors into customers.

= How many prizes can I add to the wheel? =
Unlimited! Popular configurations include 6-8 segments for balanced gameplay, 12+ segments for variety, or mixed high-value and smaller prizes.

= What types of rewards can I offer? =
Percentage discounts (5-50% off), fixed amounts ($5-$25 off), free shipping, BOGO deals, free samples, loyalty points, and custom messages.

= When does the spin wheel appear? =
Complete control with triggers: page load, time delay, scroll percentage, exit intent, specific pages, or for specific user types.

= Can I target specific visitors? =
Yes! Target by geography, device type, browser, traffic source, user behavior, and time-based rules.

= How do I prevent spam or abuse? =
Built-in protection includes IP restrictions, email validation, rate limiting, fraud detection, honeypot protection, and moderation options.

= Is Ultimate Spin Wheel compatible with my theme and plugins? =
Yes! Works with all WordPress themes and major plugins: WooCommerce, Elementor, Divi, WPBakery, Gutenberg, WPML, Yoast SEO, and caching plugins.

= Can I customize the wheel design to match my brand? =
Absolutely! Custom colors, gradients, logos, images, typography, animation speeds, sound controls, and mobile-responsive layouts.

= Does it work on mobile devices? =
Yes! Built mobile-first with touch-optimized interactions that adapt to all screen sizes.

= Will it slow down my website? =
No! Optimized with lazy loading, compressed assets, CDN-ready files, and trigger-based loading. Average impact: <0.1 seconds.

= Is it GDPR compliant? =
Yes! Includes GDPR compliance tools, cookie consent integration, data export/deletion, privacy policy integration, and explicit consent mechanisms.

= Do I need coding skills? =
Not at all! Features include drag-and-drop builder, pre-made templates, one-click wizard, and visual customization tools.
